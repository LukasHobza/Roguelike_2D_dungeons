# Roguelike_2D_dungeons

Cílem projektu je vytvořit desktopovou 2D hru s využitím přívětivého UI/UX. Hlavní herní mechanikou je generování náhodných map ve stylu bludiště. Herním cílem hráče je projít všechna bludiště až na konec, kde dojde k závěrečnému souboji s bossem. Aby prošel do dalších bludiští, musí hráč nalézt správné dveře. Navíc hráče během jeho cesty budou chtít zastavit nepřátelé. Jejich úroveň se bude s bludišti postupně zvyšovat a proto musí hráč nepřátele ničit, aby získával předměty a herní měnu. V hlavním menu bude obchod pro vylepšení schopností (síla, výdrž, …) za herní měnu. Dále bude možnost nastavení, kde hráč volí hlasitost, rozlišení obrazovky. Při spuštění hry se hráč rovnou ocitne v prvním bludišti s jednoduchým návodem. Při prohře hráč ztrácí všechny předměty, zlato hráči zůstává. Hráč může měnit rychlost pohybu. Inventář je řešen formou mřížky, každý předmět zabírá 1 buňku. Bude řešen rozmanitý výběr předmětů a jejich kombinací (sady předmětů). Na konci každého bludiště se zobrazí krátká animace s výsledným počtem herní měny, času průchodu a počtem poražených nepřátel. Použité technologie: C# nebo C++, GitHub.
